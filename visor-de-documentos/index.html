Prompt 3 – Subida de archivos a Supabase

Prompt original:
quiero subir archivos y que se guarden en supabase para que luego pueda abrirlos cuando quiera.

Prompt pulido:
Crear un módulo de subida de archivos que permita cargar PDFs, imágenes, vídeos u otros documentos directamente al almacenamiento de Supabase.
El sistema debe validar formatos, subir el archivo, guardarlo con metadatos y asociarlo al usuario autenticado.

uploadFile: async (file) => {
    const timestamp = new Date().getTime();
    const safeName = file.name.replace(/[^a-zA-Z0-9.\-_]/g, '_');
    const path = `${timestamp}_${safeName}`;

    app.toggleLoader(true);
    
    try {
        // 1. Subir al Storage de Supabase
        const { error: uploadErr } = await supabase.storage.from('uploads').upload(path, file);
        if (uploadErr) throw uploadErr;

        const { data: { publicUrl } } = supabase.storage.from('uploads').getPublicUrl(path);

        // 2. Guardar metadatos en Base de Datos
        const { error: dbErr } = await supabase.from('files_metadata').insert([{
            name: file.name,
            type: app.getExt(file.name),
            size: file.size,
            url: publicUrl
        }]);

        if (dbErr) throw dbErr;
        app.toast('Archivo subido correctamente', 'success');
        await app.fetchFiles(); // Actualizar vista
    } catch (err) {
        app.toast('Error: ' + err.message, 'error');
    } finally {
        app.toggleLoader(false);
    }
},

// La función fetchFiles actualiza toda la UI (tablas, gráficos, contadores) sin recargar la página
fetchFiles: async () => {
    app.toggleLoader(true);
    const { data, error } = await supabase
        .from('files_metadata')
        .select('*')
        .order('created_at', { ascending: false });

    // Actualización dinámica del DOM
    app.files = data;
    app.updateUI(); // Refresca tablas y gráficos instantáneamente
    app.toggleLoader(false);
},
