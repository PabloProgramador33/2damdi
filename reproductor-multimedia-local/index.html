Reproductor Multimedia Local

Todos los prompts que he usadolos he pulido con chatgpt antes de pasarlo a gemini ya que no quería perder intentos o que hubiese errores en mis prompts y que hiciese mas lio en gemini.

Por lo tanto he hecho mi prompt con mis palabras, chatgpt lo ha traducido a un lenguaje mas exacto para gemini y asi he ido trabajando.

Prompt 1 – Inicio del proyecto

Prompt original:
quiero hacer un reproductor de musica local, que lea mis canciones desde el pc y las reproduzca, y que se vea todo bonito en una sola pagina html. que tenga un calendario tambien para poner canciones a dias concretos.

Prompt pulido:
Crear un Reproductor Multimedia Local con un diseño limpio y responsivo en HTML, CSS y JavaScript.
Permitir cargar canciones desde archivos locales.
Incluir un calendario interactivo donde se pueda asignar música a días concretos.
Interfaz visual moderna y minimalista, optimizada para escritorio y móvil.

 

Prompt 2 – Carga y gestión de música

Prompt original:
quiero que se pueda arrastrar canciones todas a la vez y que se carguen en la app, que me deje crear una playlist con ellas y ver sus nombres.

Prompt pulido:
Implementar sistema de carga múltiple de archivos mediante arrastrar y soltar (drag & drop).
Permitir importar varias canciones locales simultáneamente.
Mostrar nombre, duración y carátula (si existe).
Crear listas de reproducción personalizadas y asociarlas a días específicos.

Prompt 3 – Calendario y estados de ánimo

Prompt original:
en el calendario quiero poner como me siento ese dia, tipo feliz, triste, cansado. y que tambien me deje poner un resumen del dia o una nota.

Prompt pulido:
Diseñar un calendario emocional interactivo integrado al reproductor.
Posibilidad de seleccionar un día y asignar un estado de ánimo (feliz, relajado, triste, motivado, etc.).
Añadir un resumen del día y notas personales.
Guardar los datos localmente con fecha y hora para crear un registro visual del estado emocional del usuario.

Prompt 4 – Visualizador y experiencia de escucha

Prompt original:
quiero que mientras suena la musica se vea un visualizador con ondas o barras que se muevan al ritmo, que parezca profesional.

Prompt pulido:
Añadir un visualizador de audio dinámico sincronizado con la música.
Utilizar la API de audio para generar ondas o barras animadas al ritmo del sonido.
Adaptar el diseño del visualizador al tema elegido (oscuro o claro).
Integrar animaciones suaves para ofrecer una experiencia inmersiva.

Prompt 5 – Configuración y personalización

Prompt original:
poner una seccion de configuracion para cambiar colores, fuente y modo oscuro, que quede guay.

Prompt pulido:
Crear una sección de configuración donde el usuario pueda cambiar el tema visual (modo claro, oscuro, dorado, etc.), seleccionar tipografía personalizada y ajustar parámetros visuales del reproductor.
Todo sin perder coherencia con la identidad visual de MiAudio.

Prompt 6 – Estadísticas y seguimiento

Prompt original:
quiero ver graficas con mi progreso, cuantas canciones escuche cada semana, y si estuve feliz o triste.

Prompt pulido:
Implementar un panel de estadísticas que muestre información visual sobre el uso de MiAudio.
Gráficas diarias, semanales y mensuales de reproducción.
Promedios de estado de ánimo y número de notas por día.
Datos simulados de meses anteriores para mostrar progreso.
Visualización clara mediante barras, líneas y colores emocionales.

Prompt 7 – Notas rápidas e historial

Prompt original:
añadir un boton para escribir notas durante los descansos o mientras escucho musica, y que se guarden con fecha.

Prompt pulido:
Incorporar un sistema de notas rápidas.
Botón “Notas” visible durante la sesión para registrar pensamientos o ideas.
Guardado automático con fecha y hora exacta.
Sección “Historial de Notas” donde consultar las entradas anteriores.

Prompt 8 – Reinicio y administración

Prompt original:
poner una opcion para borrar todo y empezar desde cero, pero sin romper el diseño.

Prompt pulido:
Añadir opción de reinicio completo.
Restablecer playlists, notas, estados de ánimo y estadísticas.
Mantener la estructura visual y la configuración de temas.
Confirmación previa antes de borrar datos.

Prompt 9 – Diseño adaptable y profesional

Prompt original:
quiero que en movil se vea mas ordenado el menu de arriba, que todo tenga su sitio y se vea bien.

Prompt pulido:
Optimizar el diseño responsive para dispositivos móviles.
Reorganizar el menú superior con iconos claros y jerarquía visual.
Mantener accesibles todas las funciones (calendario, playlist, estadísticas, notas).
Asegurar una experiencia fluida y profesional tanto en móvil como en escritorio.
