Pomodoro

Todos los prompts que he usadolos he pulido con chatgpt antes de pasarlo a gemini ya que no quería perder intentos o que hubiese errores en mis prompts y que hiciese mas lio en gemini.

Por lo tanto he hecho mi prompt con mis palabras, chatgpt lo ha traducido a un lenguaje mas exacto para gemini y asi he ido trabajando.

Prompt 1

Prompt original:
quiero hacer una app pomodoro con gemini, que tenga un temporizador de 25 min y despues 5 de descanso.
que tenga botones para iniciar para parar y reset.
que se vea bonito y tranquilo y que muestre la tarea que estoy haciendo.

Prompt pulido:
Crear una app de Pomodoro usando Gemini Canvas.

  • Visualizar temporizador de 25 minutos y descanso de 5 minutos.

  • Botones de iniciar, pausar y reset.

  • Animaciones suaves y estilo relajante.

  • Mostrar la tarea activa junto al temporizador.

Prompt 2

Prompt original:
añadir que pueda poner varias tareas y poder elegir cual es la activa.
que se vea en el canvas la lista de tareas y se actualice al cambiar.

Prompt pulido:
Añadir un sistema de tareas:

  • Lista de tareas activas dibujadas en Canvas.

  • Permitir seleccionar la tarea activa.

  • Actualizar la tarea activa en tiempo real mientras el Pomodoro corre.

Prompt 3

Prompt original:
hacer que el reloj cambie de color segun si estoy en pomodoro o descanso.
poner una barra circular que vaya llenandose.
y que avise visualmente cuando termina.

Prompt pulido:
Mejorar las animaciones del temporizador:

  • Cambiar color del temporizador según fase (Pomodoro/Descanso).

  • Agregar barra circular de progreso animada.

  • Feedback visual al terminar un Pomodoro.

Prompt 4

Prompt original:
que se guarde cuantos pomodoros hago cada dia.
que se pueda ver un historial y saber cuantos he hecho.

Prompt pulido:
Guardar el número de Pomodoros completados por día.

  • Preparar estructuras de datos para mostrar historial.

  • Permitir ver cuántos Pomodoros se han completado cada día.

Prompt 5

Prompt original:
poner un menu estadisticas donde se vea graficas de pomodoros por dia semana y mes.
que se vea en canvas y que haya datos simulados de meses anteriores.

Prompt pulido:
Añadir sección "Estadísticas" al menú:

  • Mostrar gráficas de Pomodoros por día, semana y mes.

  • Dibujo en Canvas.

  • Datos simulados para meses anteriores.

  • Mantener estilo relajante y colores coherentes con el resto de la app.

Prompt 6

Prompt original:
quiero un calendario donde pueda poner tareas para otros dias.
que pueda marcar completadas o pendientes.
que se vea al hacer click en el dia y se integre con estadisticas.

Prompt pulido:
Crear un calendario interactivo en Canvas:

  • Permitir añadir tareas para fechas futuras.

  • Marcar tareas completadas o pendientes.

  • Mostrar tareas diarias al hacer click en la fecha.

  • Integrar con el menú de Estadísticas.

Prompt 7

Prompt original:
graficas de progreso, por dia semana y mes.
poner datos de ejemplo simulados para ver como queda.

Prompt pulido:
Dibujar gráficas de progreso en Canvas:

  • Tipo día: pomodoros completados hoy vs promedio histórico.

  • Tipo semana: total de pomodoros semanales.

  • Tipo mes: comparación con meses anteriores.

  • Valores simulados para probar diseño.

Prompt 8

Prompt original:
poner un apartado de notas para escribir ideas durante los descansos.
que se guarden con fecha y hora.
y que se vea el historial en canvas.

Prompt pulido:
Añadir sección "Notas" al menú:

  • Permitir escribir pensamientos o ideas durante descansos.

  • Guardarlas automáticamente con fecha y hora.

  • Mostrar historial de notas en Canvas.

  • Integrar notas con el menú principal de la app.

Prompt 9

Prompt original:
hacer que la app se vea bien en movil.
que el menu quede ordenado y se vea profesional.
todas las funciones accesibles sin desorden.

Prompt pulido:
Adaptar la app a móviles:

  • Escalar Canvas automáticamente según pantalla.

  • Organizar el menú verticalmente de forma estética.

  • Todas las funciones (Tareas, Estadísticas, Calendario, Notas) accesibles sin desorden.

  • Mantener estilo profesional y limpio.

Prompt 10

Prompt original:
integrar todo lo de arriba en gemini canvas.
temporizador pomodoro, tareas, menu estadisticas calendario notas,
graficas de progreso y notas con historial, que todo funcione bien junto.

Prompt pulido:
Integrar todas las funciones en Gemini Canvas:

  • Temporizador Pomodoro

  • Selección de tareas

  • Menú con Estadísticas, Calendario y Notas

  • Gráficas de progreso (día, semana, mes) con datos simulados

  • Notas rápidas con historial

  • Modo móvil optimizado

  • Animaciones y feedback visual consistentes
