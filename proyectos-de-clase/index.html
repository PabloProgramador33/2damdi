1. Calculadora Básica

Desarrollo de una calculadora de escritorio en Java que permite realizar operaciones aritméticas básicas: suma, resta, multiplicación y división. Implementa interfaz gráfica con Swing, manejo de eventos y validaciones de entrada. Con este proyecto reforcé mis conocimientos en programación orientada a objetos y GUI en Java.

import java.util.Scanner;

public class Calculadora {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Bienvenido a la calculadora");
        System.out.print("Introduce el primer número: ");
        double num1 = sc.nextDouble();
        System.out.print("Introduce el segundo número: ");
        double num2 = sc.nextDouble();

        System.out.println("Selecciona operación: 1-Suma 2-Resta 3-Multiplicación 4-División");
        int opcion = sc.nextInt();
        double resultado = 0;

        switch(opcion) {
            case 1: resultado = num1 + num2; break;
            case 2: resultado = num1 - num2; break;
            case 3: resultado = num1 * num2; break;
            case 4: 
                if(num2 != 0) resultado = num1 / num2; 
                else System.out.println("Error: División entre 0");
                break;
            default: System.out.println("Opción no válida");
        }

        System.out.println("Resultado: " + resultado);
        sc.close();
    }
}

2. Comprobador de Emails

Aplicación que valida la sintaxis de correos electrónicos introducidos por el usuario. Utiliza expresiones regulares y lógica condicional para detectar errores comunes en emails. Aprendí a manejar validaciones y excepciones en Java, así como a crear algoritmos eficientes para comprobación de datos.

import java.util.Scanner;
import java.util.regex.*;

public class ComprobadorEmail {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce un email: ");
        String email = sc.nextLine();

        String regex = "^[\\w-\\.]+@([\\w-]+\\.)+[\\w-]{2,4}$";
        Pattern pattern = Pattern.compile(regex);
        Matcher matcher = pattern.matcher(email);

        if(matcher.matches()) {
            System.out.println("Email válido");
        } else {
            System.out.println("Email inválido");
        }

        sc.close();
    }
}

3. Gestor de Contraseñas

Herramienta para generar, almacenar y validar contraseñas seguras. Incluye opciones de longitud, complejidad y cifrado básico para proteger los datos. Este proyecto me permitió practicar seguridad informática y técnicas de cifrado sencillo en Java, así como buenas prácticas de almacenamiento de información sensible.

import java.util.Random;
import java.util.Scanner;

public class GeneradorContrasena {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce longitud de la contraseña: ");
        int longitud = sc.nextInt();

        String chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789!@#$%";
        Random rand = new Random();
        StringBuilder pass = new StringBuilder();

        for(int i = 0; i < longitud; i++) {
            pass.append(chars.charAt(rand.nextInt(chars.length())));
        }

        System.out.println("Contraseña generada: " + pass);

        sc.close();
    }
}

5. Gestión de Empleados

Sistema de gestión de empleados para una empresa ficticia. Permite alta, baja, modificación y búsqueda de empleados, además de calcular salarios y turnos. Utiliza arrays y colecciones, validación de datos y estructura modular en Java. Aprendí a organizar un programa con varias clases y mantener la integridad de los datos.

import java.util.ArrayList;
import java.util.Scanner;

public class GestionEmpleados {
    public static void main(String[] args) {
        ArrayList<String> empleados = new ArrayList<>();
        Scanner sc = new Scanner(System.in);

        do {
            System.out.println("1-Agregar empleado 2-Mostrar empleados 3-Salir");
            opcion = sc.nextInt();

            switch(opcion) {
                case 1:
                    System.out.print("Nombre del empleado: ");

                    empleados.add(nombre);
                    System.out.println("Empleado agregado");
                    break;
                case 2:
                    System.out.println("Empleados:");

                default: System.out.println("Opción no válida");
            }
        } while(opcion != 3);

        sc.close();
    }
}

6. Concesionario de Vehículos

Aplicación que permite gestionar un concesionario: registro de vehículos, ventas, clientes y stock. Se implementa interfaz gráfica con Swing, estructuras de datos y consultas internas. Este proyecto me ayudó a practicar programación orientada a objetos y lógica de gestión de inventario.

import java.util.ArrayList;
import java.util.Scanner;

class Vehiculo {
    String marca;
    String modelo;
    int año;
    double precio;

    Vehiculo(String marca, String modelo, int año, double precio) {
        this.marca = marca;
        this.modelo = modelo;
        this.año = año;
        this.precio = precio;
    }

    public String toString() {
        return marca + " " + modelo + " (" + año + ") - " + precio + "€";
    }
}

public class Concesionario {
    public static void main(String[] args) {
        ArrayList<Vehiculo> stock = new ArrayList<>();
        Scanner sc = new Scanner(System.in);
        int opcion;

        do {
            System.out.println("\n1-Agregar vehículo 2-Mostrar stock 3-Salir");
            opcion = sc.nextInt();
            sc.nextLine();

            switch(opcion) {
                case 1:
                    System.out.print("Marca: "); String marca = sc.nextLine();
                    System.out.print("Modelo: "); String modelo = sc.nextLine();
                    System.out.print("Año: "); int año = sc.nextInt();
                    System.out.print("Precio: "); double precio = sc.nextDouble(); sc.nextLine();
                    stock.add(new Vehiculo(marca, modelo, año, precio));
                    System.out.println("Vehículo agregado al stock");
                    break;
                case 2:
                    System.out.println("Stock disponible:");
                    for(Vehiculo v : stock) System.out.println(v);
                    break;
                case 3: System.out.println("Saliendo..."); break;
                default: System.out.println("Opción no válida");

7. Juego de Bingo

Juego de bingo interactivo en Java para consola o GUI, con generación de cartones, sorteo aleatorio de números y control de ganador. Este proyecto mejoró mis conocimientos en aleatoriedad, bucles y manejo de arrays multidimensionales.

import java.util.Random;
import java.util.HashSet;

public class Bingo {
    public static void main(String[] args) {
        Random rand = new Random();
        HashSet<Integer> carton = new HashSet<>();

        while(carton.size() < 5) { // generar 5 números aleatorios para el cartón
            int numero = rand.nextInt(75) + 1;
            carton.add(numero);
        }

        System.out.println("Tu cartón de bingo: " + carton);

        System.out.println("Sorteando 5 números...");
        for(int i=0; i<5; i++) {
            int sorteo = rand.nextInt(75) + 1;
            System.out.println("Número sorteado: " + sorteo);
            if(carton.contains(sorteo)) System.out.println("¡Coincidencia!");
        }
    }
}

9. Cifrado de Mensajes

Programa que permite cifrar y descifrar mensajes usando algoritmos sencillos como César o sustitución. Este proyecto me permitió profundizar en manipulación de cadenas y seguridad básica de datos.

import java.util.Scanner;

public class CifradoCesar {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce mensaje: ");
        String mensaje = sc.nextLine();
        System.out.print("Introduce clave (número de desplazamiento): ");
        int clave = sc.nextInt();

        StringBuilder cifrado = new StringBuilder();
        for(char c : mensaje.toCharArray()) {
            if(Character.isLetter(c)) {
                char base = Character.isUpperCase(c) ? 'A' : 'a';
                c = (char)((c - base + clave) % 26 + base);
            }
            cifrado.append(c);
        }

        System.out.println("Mensaje cifrado: " + cifrado);
        sc.close();
    }
}

10. Calculadora de IMC

Aplicación para calcular el índice de masa corporal (IMC) y clasificarlo según rangos de salud. Incluye validaciones de entrada y mensajes personalizados. Practiqué la lógica condicional y manejo de números decimales en Java.

import java.util.Scanner;

public class IMC {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce tu peso en kg: ");
        double peso = sc.nextDouble();
        System.out.print("Introduce tu altura en metros: ");
        double altura = sc.nextDouble();

        double imc = peso / (altura * altura);
        System.out.println("Tu IMC es: " + imc);

        if(imc < 18.5) System.out.println("Bajo peso");
        else if(imc < 25) System.out.println("Peso normal");
        else if(imc < 30) System.out.println("Sobrepeso");
        else System.out.println("Obesidad");

        sc.close();
    }
}

13. Juego de Trivia de Conocimientos

Juego de preguntas y respuestas con múltiples niveles de dificultad. Incluye puntuaciones, temporizador y selección aleatoria de preguntas. Me permitió practicar algoritmos, estructuras de control y diseño de juegos educativos en Java.

import java.util.Scanner;

public class Trivia {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String[][] preguntas = {
            {"¿2+2=?", "4"},
            {"¿Capital de España?", "Madrid"},
            {"¿Java es un lenguaje de programación? (sí/no)", "sí"}
        };

        int puntuacion = 0;
        for(String[] p : preguntas) {
            System.out.println(p[0]);
            String respuesta = sc.nextLine();
            if(respuesta.equalsIgnoreCase(p[1])) {
                System.out.println("¡Correcto!");
                puntuacion++;
            } else {
                System.out.println("Incorrecto, la respuesta correcta es: " + p[1]);
            }
        }

        System.out.println("Puntuación final: " + puntuacion + "/" + preguntas.length);
        sc.close();
    }
}
